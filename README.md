# Nemo WebP

WebP image format support for thumbnails in Nemo.

Package description for the AUR repository. 

## Installation

```bash
sudo pacman -S nemo-webp
```

## Description

Install a config file for Nemo to use the WebP thumbnailer.

## Dependencies

- libwebp
- nemo
